# 认识秋天的树叶

- 时间：2018年11月10日 星期六
- 天气：晴
- 地点：小区里

```
今天爸爸说要带我去认识小区里的植物，我带上了采集树叶的盒子和剪刀，就从我们家楼下开始了。我拿着爸爸的手机，打开一款叫《形色》的APP，对着植物的花或叶子，拍一下照片，就可以知道它是哪一种植物了。
下面来请跟我一起看看今天的收获吧。
```
